**功能说明:**



系统具有多种角色，管理员，操作人员和住户 ，不同的角色进入后功能不一样

管理员 ： 用户管理，角色管理 菜单管理 小区管理 等

操作人员  电费管理 账单信息查看 用户管理  个人信息

住户  缴费  查看账单  个人信息
